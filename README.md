# kylyp (rust-web qq群:661633549)

A new Forum !

The project use rust, rocket, diesel, postgresql, and responsive design to build a new Forum.

### Start

```bash
$ cargo install diesel_cli
$ diesel setup
$ diesel migration run
$ cargo run
```
